Setting up this live developer journal with a few articles, code snippet syntax highlighting and a CSS reset

19 JUL 2019

Instead of waiting for this website to be 'ready' before it is published, I have decided to publish it as a raw HTML site without all of the bells and whistles. Everything is a work in progress.

The first thing I did was set up Github pages site to host this developer journal website on. Then I wrote an article about it and added a link to it on the home page.

As this website build is not my priority right now, I'm usually making changes to make my life easier, or to solve something that's pretty annoying. The first thing that bothered me was the state of my code snippets. In raw HTML, they don't look very good. So I installed Prism, which is a syntax highlighting tool for code snippets. Instructions for doing this are on the prism website. You can change the styles in the prism.css file to suit your branding better. I changed my code snippets so that they were fullscreen and had a darker background
