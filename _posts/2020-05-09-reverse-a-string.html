Problem Statemest

Reverse the provided string.

You may need to turn the string into an array before you can reverse it.

Your result must be a string.

Solution

Not much to this one. When I didn't include the quotes in .join(), the string had commas between each letter in the string. The quotes joined them with nothing in between the letters.


function reverseString(str) {
  return Array.from(str).reverse().join('');
}

reverseString("hello");
